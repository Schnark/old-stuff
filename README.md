This repository contains all my "old" files, taken from an old backup. Some of these files are not listed below, but most of them, that is, most files I created between January 1994 and November 2007 are included. There are also some other, newer files with code that hasn't been published before, and which I (probably) will no longer work on.

Please note the following:
* All comments and all text is in German.
* The old code uses code page 850 as text encoding.
* Most filenames were created when they had to conform to the 8+3 scheme.
* Most DOC files were created with Microsoft Word 2.
* Some files are omitted for privacy or copyright reasons.
* If something doesn't work, it could be due to a missing library that isn't included, because it relies on some internals of MS-DOS 6, or because it never worked at all.

If I decide to abandon a yet unfinished and unpublished program, I might add it to this repository, and I might also add some more explanations to the programs, but I wo'n't otherwise update anything in here.

The following lists state the contents of each folder, including omitted files and with timestamps, to give you an idea about how old the files really are. (Of course, in some cases the files were created much earlier, and only edited again later.)
